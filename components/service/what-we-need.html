What We Need to Confirm Quickly

For a fast confirmation, we prefer to receive in one message: date, time window, desired duration, and the setting (e.g., dinner/casino/club/event/hotel). Dress code or style preference is optional, but helps us create the best match.

Checklist

  • Date + time window (start time or start range)
  • Setting/location (name or neighborhood is enough)
  • Desired duration
  • Dress code/style (optional)
  • Language (NL/EN)

Communication stays intentionally limited and professional: only what’s needed to organize it properly.
